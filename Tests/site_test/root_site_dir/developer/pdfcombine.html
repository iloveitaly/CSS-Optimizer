PDFCombine

What is it?

PDFCombine is a simple command line utility that allows you to combine the pages of PDF documents. Unlike some other PDF combining solutions PDFCombine keeps your PDF documents intact: if your PDF document was searchable before, it will still be searchable; if text was selectable before, it will still be selectable. Additionally, PDFCombine does not add any extra bloat to your PDF documents. OS X 10.4.x is required for PDFCombine to run since the PDFKit framework is used in this application.

As of now, there is no user interface for this application. If there is enough demand for one I'll create it, so if you want a user interface for PDFCombine, Let Me Know!
